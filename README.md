# memoryRank

Ein TypeScript Node.js Projekt.

## Installation

```bash
npm install
```

## Entwicklung

```bash
# Code kompilieren
npm run build

# Projekt starten
npm start

# Development-Modus (ohne Kompilierung)
npm run dev

# TypeScript im Watch-Modus
npm run watch
```

## Projektstruktur

```
memoryRank/
├── src/           # TypeScript-Quellcode
├── dist/          # Kompilierter JavaScript-Code (wird generiert)
├── package.json   # Projekt-Dependencies und Scripts
├── tsconfig.json  # TypeScript-Konfiguration
└── README.md      # Diese Datei
```
